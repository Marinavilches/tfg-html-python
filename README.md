# Anexos HTML del Trabajo Fin de Grado

Este repositorio contiene los archivos HTML generados a partir de los notebooks de Python utilizados como anexo digital del Trabajo Fin de Grado. En ellos se recoge el desarrollo técnico del análisis realizado, desde la preparación inicial de los datos hasta la aplicación de modelos de aprendizaje automático.

## Archivos incluidos

1. [Lectura y preparación del dataset](01_Lectura_Preparacion_Dataset.html)
   En este archivo se recoge la lectura de los datos originales, la revisión inicial de las tablas disponibles y la preparación de la base de datos utilizada en el análisis.

2. [Tratamiento y calidad del dato](02_Tratamiento_Calidad_Dato.html)
   En este apartado se revisan aspectos relacionados con la calidad del dato, como valores ausentes, duplicados, formatos de variables e inconsistencias detectadas durante el proceso.

3. [Análisis exploratorio](03_Analisis_Exploratorio.html)
   Este archivo incluye el análisis descriptivo y visual de las variables, con el objetivo de comprender mejor la estructura del dataset e identificar patrones relevantes.

4. [Modelos de regresión](04A_Modelos_Regresion.html)
   En este documento se desarrollan los modelos de regresión aplicados en el trabajo, junto con su evaluación e interpretación de resultados.

5. [Modelos de clasificación y aplicación](04B_Modelos_Clasificacion_y_Aplicacion.html)
   Este archivo recoge la construcción y evaluación de los modelos de clasificación, así como su aplicación al contexto del análisis realizado.
